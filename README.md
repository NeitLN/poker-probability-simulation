# 🎲 Poker Probability Simulation & EV Analysis
> 📚 This project is developed as a Final-Term Project for the Discrete Mathematics course at Van Lang University.
> 
> Probabilistic analysis of Poker using combinatorics, Monte Carlo simulation, and Expected Value (EV) to study decision-making and convergence behavior.

---

## 📌 Overview

This project applies concepts from **Discrete Mathematics** to analyze Poker through:

- 🔢 Combinatorics  
- 📊 Probability Theory  
- 🎲 Monte Carlo Simulation  
- 💰 Expected Value (EV)  
- 📈 Statistical Analysis  

---

## 🎯 Objectives

- Compute theoretical probabilities of 5-card Poker hands  
- Simulate Poker using Monte Carlo methods  
- Compare theoretical and experimental results  
- Analyze convergence behavior (Law of Large Numbers)  
- Evaluate Expected Value (EV) for decision-making  

---

## 🧮 Mathematical Concepts

- **Combinations:** C(n, k)  
- **Sample space:** C(52, 5)  
- **Probability distributions**  
- **Expected Value (EV)**  
- **Error analysis**
